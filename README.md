Esta é uma API RESTful simplificada para gerenciamento de catálogos de jogos, desenvolvida para demonstrar o fluxo básico de operações CRUD (Create, Read, Update, Delete) e a comunicação entre sistemas via protocolo HTTP.
Função de cada methods:
post: adiciona um novo elemento na lista,
delete: deleta um elemento na lista,
put: edita um dos elementos da lista,
get: pega um elemento da lista ou todos.
